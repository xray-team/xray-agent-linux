### Description
A simple lightweight service for collecting metrics.

### Compatibility
Tested with:
* ubuntu 22.04 LTS
* ubuntu 20.04 LTS
* ubuntu 18.04 LTS
* debian 10
* debian 9
* centos 8
* centos 7

### Packages
The deb and rpm packages are available at the [Releases page](https://github.com/xray-team/xray-agent-linux/releases/latest).
